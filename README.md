## üèÉ About The Game
**Royal Run 3D** is a fast-paced endless runner developed in Unity. 

Inspired by genre classics like *Subway Surfers*, players control a Prince sprinting across a perilous sky-bridge. You must react quickly to dodge oncoming objects, barrels, and crates while collecting coins and power-ups. The game features a high-score system and increasing difficulty as you survive longer against the clock.

## ‚ú® Key Features
* **Endless Running Mechanic:** The path generates continuously with increasing speed.
* **Dynamic Obstacles:**
    * **Oncoming Crates:** Massive objects flying toward the player that must be dodged.
    * **Static Barriers:** Barrels and fences blocking the path.
* **Power-Up System:** Collect **Speed Ups** to dash through the level and maximize your distance.
* **Score & Economy:** Gather gold coins to boost your high score.
* **High-Quality Assets:** 3D stylized character models, smooth animations, and immersive sound effects.
